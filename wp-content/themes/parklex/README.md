# Theme `parklex`

Tema FSE (Full Site Editing) para WordPress.

## Estilos

Los estilos se compilan con PostCSS desde `src/` en cuatro capas:

- `framework/` — reset, layout, variables CSS y utilidades
- `main/` — estilos de página y secciones
- `blocks/` — overrides de bloques core de WordPress
- `editor.postcss.scss` — estilos exclusivos del editor de bloques
