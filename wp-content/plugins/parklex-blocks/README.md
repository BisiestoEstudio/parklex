# Plugin `parklex-blocks`

Plugin de bloques Gutenberg personalizados construidos con `@wordpress/scripts`.

## Bloques

Los bloques viven en `src/blocks/{block-name}/` y se registran automáticamente desde `build/blocks/`.

## Componentes reutilizables

- `MediaPicker` — selector de imagen o vídeo
- `LinkPicker` — selector de enlace interno o externo
